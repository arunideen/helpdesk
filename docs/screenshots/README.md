# Screenshots

Add application screenshots to this directory. The README references the following files:

| Filename | Page |
|----------|------|
| `login.png` | Login page |
| `ticket-list.png` | Ticket dashboard with filters |
| `create-ticket.png` | Create new ticket form with attachments |
| `ticket-detail.png` | Ticket detail view with comments and sidebar |
| `users.png` | User management page (admin) |
| `settings.png` | System settings page (admin) |

## How to Capture

1. Run the app: `docker compose up -d`
2. Open http://localhost:5173
3. Log in as admin (`admin@company.com` / `admin123`)
4. Navigate to each page and take a screenshot
5. Save the screenshots to this directory with the filenames above
